# My ML Project

## 📖 프로젝트 소개
**My ML Project**는 머신러닝을 활용하여 해산물 가격을 예측하는 서비스입니다.  
소비자에게 최적의 구매 시점을 제공하여 경제적 이득을 제공합니다.

---

## 🚀 주요 기능
1. **데이터 전처리**: 기상 데이터, 검색 트렌드, 수산물 가격 데이터를 정리 및 분석.
2. **모델 학습**: 머신러닝 모델을 통해 가격 예측 수행.
3. **결과 시각화**: 예측된 데이터를 그래프 및 표로 제공.
